Микросервис cd_generator отвечает за статистический блок - генерирует и обновляет статистику
по вакансиям, отслеживает вакансии, управляет экзаменами и доступом к ним.
Является частью проекта CheckDev.

Применяемые технологии:
SpringBoot,
Liquibase,
Spring Security,
Spring Data JPA
PostgreSQL
Java 17 и выше
Maven
REST API
Jenkins
HeadHunter API
